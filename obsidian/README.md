# Obsidian Vault — hosted-aiOS

Öppna denna mapp som en vault i Obsidian.

## Struktur

| Mapp | Innehåll |
|------|----------|
| `00-dashboard.md` | Startsida — projektstatus |
| `01-planning/` | Alla planer + masterplan |
| `02-decisions/` | ADR:er |
| `03-meetings/` | Mötesanteckningar |
| `04-research/` | Research + teknik-spaning |
| `05-building/` | Byggloggar |
| `06-personal/` | Personliga anteckningar (en mapp per person) |
| `templates/` | Mallar för nya noter |

## Snabbstart

1. Installera [Obsidian](https://obsidian.md)
2. Öppna denna mapp som en vault
3. Aktivera templates: Settings → Templates → Template folder location = `templates`
4. Öppna `00-dashboard.md` — där börjar du
