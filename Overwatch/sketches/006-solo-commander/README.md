# Variant: Shadow Commander

## Design stance
Jinwoo's Shadow Army — agenter är soldater med rank (S/A/B), order deployas, rapporter inkommer. Mörk fantasy-krigsrumskänsla.

## Key choices
- Layout: Sidebar med soldatlista + huvudvy med transmissioner
- Typography: JetBrains Mono för kommandon/rapporter, Inter för chatt
- Color: Violet-dominant (#8b5cf6), guld rank, "MANA" för VRAM
- Interaction: Soldater klickbara, DEPLOY-knapp, order-block i transmissioner

## Trade-offs
- Strong at: Mest immersiv Solo Leveling-upplevelse, rankad hierarki, fantasy-tema
- Weak at: Minst praktisk för seriöst arbete, "soldat"-metafor kan bli tröttsam

## Best for
Maximal Solo Leveling-immersion — användare som vill ha fantasy-krigsrummet fullt ut.
