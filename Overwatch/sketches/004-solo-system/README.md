# Variant: Solo System

## Design stance
Solo Leveling "System"-gränssnitt — blå neon-rutor, stat-block, quest-logg, rankad shadow army.

## Key choices
- Layout: Chat vänster + Sidepanel höger (player stats, quests, agents, GPU)
- Typography: Inter body, JetBrains Mono för all data/statistik
- Color: #06060c bakgrund, system-blue #6366f1 neon, guld #f59e0b för rank
- Interaction: System-meddelanden som popuprutor, EXECUTE-knapp

## Trade-offs
- Strong at: Solo Leveling-känsla, gaming-atmosfär, rankad agenthierarki
- Weak at: Mindre chattutrymme, sidepanel kan kännas game-UI över seriös app

## Best for
Fans av Solo Leveling — vill ha System-upplevelsen med stat-block och quest-logg.
