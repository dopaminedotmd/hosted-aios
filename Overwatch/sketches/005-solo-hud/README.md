# Variant: Solo HUD

## Design stance
Gaming HUD — HP/MP-bars, inventory-grid för agenter, command-prompt. Som att vara inne i spelet.

## Key choices
- Layout: Inventory vänster (2x grid) + Chat center + Stats höger
- Typography: JetBrains Mono dominant, Inter för body
- Color: Lila/violett system-färger, guld rank, HP/MP bars i toppbaren
- Interaction: Agenter som inventory-slots, command-prompt med > prefix

## Trade-offs
- Strong at: Spelkänsla maxad, HP/MP ger direkt systemstatus, inventory-grid är taktilt
- Weak at: Kan kännas för mycket "spel" för produktivt arbete, HP/MP är kosmetiska (inte riktiga metrics)

## Best for
Gamingvana användare — vill ha Solo Leveling RPG-känsla fullt ut.
